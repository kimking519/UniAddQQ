## 📌 重要说明：平台限制

**QQ 临时会话的固有限制：**

| 功能 | 实现方式 | 可行性 |
|------|---------|--------|
| 打开聊天窗口 | `wpa.qq.com/msgrd` | ✅ 可行 |
| 自动发送消息 | Web 协议 | ❌ 不可能 |
| 发送消息给陌生人 | 官方 QQ Bot API | ❌ 仅限好友/频道成员 |
| 发送消息给陌生人 | 第三方协议 (OneBot) | ❌ 需同群或有好友关系 |

**结论：** 网页工具只能**打开聊天窗口**，无法自动发送消息。这不是 bug，是 QQ 平台的安全限制。
